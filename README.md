## PayTM

- It's a very basic MERN app that handel db transections 
- simple crud operation


## Installation



```bash
git clone
#connect to your own mongobd in /paytm/backend/db.js
```

## start Backend
```bash
cd backend
npm i
node index.js

```

## start frontend
```bash
#opne 2 terminal

cd frontend
npm i
npm run dev

```
Try it out on you localhost:3000 port
